# PRD Directory

This directory contains active and recent Product Requirements Documents for `themion`.

## Start here

- Start PRD authoring with [PRD_AUTHORING_GUIDE.md](PRD_AUTHORING_GUIDE.md).
- `PRD_AUTHORING_GUIDE.md` is the canonical guide for PRD writing and update rules in this repository.
- For the full PRD index, use [docs/README.md](../README.md#product-requirements-documents-prds).
- Older PRDs 001–050 live under [archive/](archive/).
- Before authoring a PRD, read the most recent 2–3 PRDs in this directory and match their structure and voice.

## Conventions

- Filename format: `prd-NNN-<slug>.md`
- Use the next sequential PRD number.
- Keep one concern per PRD.
- Update `docs/README.md` when adding a new PRD.
